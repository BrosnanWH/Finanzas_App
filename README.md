
# Gastosmart

## Installation

Este proyecto esta escrito en python 3


Arch Linux
```bash
sudo pacman -S install python3 python-pip
```
```bash
git clone https://github.com/BrosnanWH/Finanzas_App
```
```bash
cd /gasto_smart
```
```bash
chmod + x gasto_smart.py
```
```bash
./gasto_smart.py
```

Debian Based
```bash
sudo apt install python3 python-pip
```

Windows

```powershell
python3 gasto_smart.py
```
